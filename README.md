# Particle_Sand_Box

Particle_Sand_Box is an interactive web project built using HTML, CSS, and JavaScript. It features a customizable particle animation sandbox with real-time controls and a dynamic feedback system using DOM manipulation.

## Features
- Interactive particle animation
- Adjustable particle count, size, speed, and color
- Connect line effects between particles
- Mouse repel interaction
- Wrap edge functionality
- Dynamic feedback form
- Form validation using JavaScript
- Live feedback counter updates

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Canvas API

## How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
3. Enjoy the interactive particle sandbox
